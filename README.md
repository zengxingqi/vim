# vim

> 一款基于vue2全家桶+koa2+mongoose+socket等技术栈的webapp即时聊天应用

## Build Setup

``` bash
# 安装依赖，推荐 yarn
npm install
yarn

# 启动dev环境
npm run dev

# 启动server socket环境
npm run server

# 并行启动dev环境和server socket环境
npm start

# 打包test代码，测试环境
npm run test:build

# 打包prod代码，线上环境
npm run prod:build

# 打包test代码和查看分析报告
npm run test:build
```
